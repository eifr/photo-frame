# Photo Frame

A self-hosted digital photo frame PWA that displays photos, videos, and GIFs sent via a Telegram bot.

## Setup

1. Open `index.html` in a browser
2. Enter your **Telegram Bot Token** (get one from [@BotFather](https://t.me/BotFather))
3. Enter your **Telegram Chat ID**
4. The app will start receiving and displaying media from that chat

## Features

- Full-screen slideshow with transition effects
- Supports photos, videos, and GIFs
- Per-media duration control (1 hour, 1 day, 1 week, forever)
- Weather overlay via Open-Meteo API
- Works offline as a Progressive Web App
- Landscape orientation optimized
- Auto-cleanup of expired media
- Message overlay support
